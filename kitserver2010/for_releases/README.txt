Quick first time installation:
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

- copy the whole "kitserver" folder into your game folder
- go to kitserver folder, and copy config-sample.txt to config.txt
- run setup.exe
- choose your game EXE (pes2010.exe usually), and press "Install"


Upgrading existing kitserver installation:
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

- go to docs folder and double-click on manual.html
- read the "Installation/Removal" of the manual. Pay attention
  to section 2.2
